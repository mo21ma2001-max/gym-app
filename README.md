## Pages
| Route | Description |
|-------|-------------|
| `/` | Home page |
| `/features` | Gym services & pricing |
| `/items` | Equipment list |
| `/items/:id` | Equipment detail |
| `/about` | About & team |
| `/contact` | Contact info |
| `/layout-demo` | EJS partials demo |
| `/*` | 404 page |

## Setup

```bash
npm install
node app.js
```

Then open → http://localhost:3000
